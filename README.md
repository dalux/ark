### Brisk
一个简单、快速、健壮的PHP扩展框架。

### 在Linux上安装： 

下载解压或克隆对应版本源码，并进入源码根目录后执行:  

./configure --with-php-config=/path/to/php-config  
make  
make install  

### Windows扩展:  

[php-brisk-1.0.3-forwindows-x86(x64).zip](https://github.com/dalux/brisk/files/2208465/php-brisk-1.0.3-all.zip)

下载对应版本，解压至扩展目录即可。
